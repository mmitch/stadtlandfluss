# Stand Land Fluss

Ein kleiner Helfer, um _Stadt Land Fluss_ zu spielen.

Man braucht weiterhin Stift und Zettel zum Spielen.
Dieses Tool hilft nur bei den Dingen, die man sonst nicht so einfach hinbekommt:

- Anfangsbuchstaben auswürfeln
- Vorschläge für Kategorien

Da Prinzip dabei ist, so einfach wie möglich zu sein.
Falls doppelte Einträge ausgewürfelt werden oder einem etwas nicht gefällt,
würfelt man einfach nochmal und fertig.

Die HTML-Seite ist vollständig (HTML + CSS + JavaScript)
und kann deshalb einfach lokal gespeichert und aufgerufen werden.

Ebenso ist es einfach möglich, eigene Kategorien zu ergänzen
oder neue Buchstabenlisten zu hinterlegen.
